What Changed in the TOEFL 2026 Format?

The TOEFL iBT changed on January 21, 2026. Here is everything you need to know.

📖

New Reading Tasks

Adaptive format with Complete the Words, Reading in Daily Life, and shorter Academic Reading passages. 30 minutes total.

✍

New Writing Tasks

Build a Sentence, Write an Email, and Write for an Academic Discussion. 23 minutes total.

⚡

Shorter & Adaptive

The test is now shorter overall. Reading uses an adaptive routing module that determines which second module you get.
